# Tradução ao Vivo — Igreja Abundante Vida

App de tradução simultânea para cultos, usando Whisper (OpenAI) para transcrição e GPT-4o mini para tradução em tempo real.

## Como acessar

👉 Acesse via GitHub Pages após publicar (veja instruções abaixo).

## Como publicar (passo a passo)

1. Crie uma conta gratuita em [github.com](https://github.com)
2. Clique em **"New repository"**
3. Nome sugerido: `traducao-culto`
4. Marque **Public**
5. Clique em **"Create repository"**
6. Clique em **"uploading an existing file"**
7. Arraste os arquivos `index.html` e `README.md`
8. Clique em **"Commit changes"**
9. Vá em **Settings → Pages → Branch: main → Save**
10. Aguarde 1-2 minutos e acesse: `https://SEU-USUARIO.github.io/traducao-culto`

## Uso

- Abra o link acima no Chrome no computador conectado ao telão
- Insira sua chave da API OpenAI
- Selecione o dispositivo de entrada (mesa de som / microfone)
- Clique em **Iniciar Tradução**
- Pressione **F11** para tela cheia no telão
